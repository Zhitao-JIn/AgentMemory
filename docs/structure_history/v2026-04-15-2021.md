## 变更说明

### 新增内容：层间通信协议

#### 1. 接口定义 (`src/protocols/interfaces.py`)
新增 4 个扩展接口支持层间通信：
- `ICommsHub` - 通信中枢接口（待实现）
- `IMemoryExtended` - 扩展记忆层接口
- `IReasoningWithComms` - 扩展推理层接口
- `IPerceptionWithComms` - 扩展感知层接口

#### 2. 消息协议 (`src/protocols/message.py`)
定义统一的层间通信消息结构：
```python
@dataclass
class Message:
    type: str           # 消息类型
    from_layer: str     # 发送方
    to_layer: str       # 接收方
    payload: Dict       # 数据负载
    timestamp: float    # 时间戳
```

#### 3. 协议模块 (`src/protocols/__init__.py`)
协议模块入口，导出所有接口和数据结构

#### 4. 新方法论实现文件

##### 感知层 (`src/perception/receive_input.py`)
```python
def receive_input(raw_input, process_fn, extract_fn, reasoning_layer, action_layer) -> str
```
实现完整流程闭环：处理输入 → 发送推理层 → 执行行动层 → 返回结果

##### 推理层 (`src/reasoning/analyze_and_generate.py`)
```python
def analyze(context, query_fn, generate_fn, memory_layer) -> str
```
实现分析 + 查询记忆 + 生成响应的完整流程

##### 记忆层 (`src/memory/update_context.py`)
```python
def update_context(data, add_fn, get_context_fn) -> Dict
def get_context(memories, recent_count) -> Dict
```
实现上下文主动维护和获取近期记忆摘要

#### 5. 各层协议修改

| 文件 | 修改内容 |
|------|----------|
| `src/perception/layer.py` | 添加 `_reasoning`, `_action` 引用，调用 `receive_input()` |
| `src/reasoning/layer.py` | 添加 `query_memory()`, `provide_feedback()`, `run()` 方法 |
| `src/memory/layer.py` | 添加 `update_context()`, `handle_query_request()`, `receive_feedback()` 方法 |
| `src/action/execute_output.py` | 支持 dict 和 str 两种类型输入 |
| `container.py` | 重新排列初始化顺序，正确注入层间引用；简化 `run()` 方法 |
