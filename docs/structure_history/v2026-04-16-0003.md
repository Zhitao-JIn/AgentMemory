## 变更说明

### 新增内容

使用 loguru 实现日志系统，支持三个日志级别：
- **ERROR**: 只输出错误信息
- **INFO**: 输出方法调用名
- **DEBUG**: 包括参数、结果等详细信息

### 使用方式

```python
# 默认 INFO 级别
agent = Container()

# 指定日志级别
agent = Container(log_level="DEBUG")  # 或 "INFO" 或 "ERROR"
```

### 日志输出格式

```
HH:mm:ss [LEVEL] [模块名] 消息
```

示例：
```
00:02:45 [INFO] [src.perception.layer] receive_input
00:02:45 [DEBUG] [src.perception.layer] receive_input - 参数：raw_input='你好'
00:02:45 [DEBUG] [src.perception.layer] receive_input - 返回："响应内容"
```
