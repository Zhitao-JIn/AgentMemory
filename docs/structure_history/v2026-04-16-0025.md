## 变更说明

### 新增国内模型提供商支持

在 Router 层中添加了主流国内 AI 提供商的支持，包括中文别名。

### 1. 新增文件

#### 国内提供商配置
| 文件 | 职责 |
|------|------|
| `src/router/providers/qwen.py` | Qwen/通义千问模型配置 |
| `src/router/providers/zhipu.py` | 智谱 AI/GLM 模型配置 |
| `src/router/providers/baichuan.py` | 百川/Baichuan 模型配置 |

#### 功能文档
| 文件 | 职责 |
|------|------|
| `docs/feature/router/domestic_providers.md` | 国内模型提供商文档 |

### 2. 修改的文件

#### `src/router/router.py`
- 新增国内模型到 `MODEL_MAP`:
  - `qwen`: qwen-max, qwen-plus, qwen-turbo
  - `zhipu`: glm-z1, glm-4, glm-3-turbo
  - `baichuan`: baichuan2-128k, baichuan2, baichuan2-turbo
  - `lingyi`: yi-34b, yi-large, yi-medium
  - `minimax`: abab6.5, abab5.5, abab4
  - `sensechat`: sensechat-5, sensechat-4, sensechat-3
- 新增中文别名到 `PROVIDER_ALIASES`:
  - `通义`, `tongyi` → qwen
  - `智谱`, `glm` → zhipu
  - `百川` → baichuan
  - `零一`, `yi` → lingyi

#### `src/router/providers/__init__.py`
- 导出 QWEN_MODELS, ZHIPU_MODELS, BAICHUAN_MODELS

#### `tests/test_router.py`
- 新增 `TestDomesticProviders` 测试类（7 个用例）

### 3. 支持的国内模型

| 提供商 | high | medium | low | 别名 |
|--------|------|--------|-----|------|
| Qwen/通义千问 | qwen-max | qwen-plus | qwen-turbo | 通义，tongyi |
| 智谱 AI/GLM | glm-z1 | glm-4 | glm-3-turbo | 智谱，glm |
| 百川/Baichuan | baichuan2-128k | baichuan2 | baichuan2-turbo | 百川 |
| 零一万物/Yi | yi-34b | yi-large | yi-medium | 零一，yi |
| MiniMax | abab6.5 | abab5.5 | abab4 | - |
| 360 智脑 | sensechat-5 | sensechat-4 | sensechat-3 | - |

### 4. 使用方式

```python
from container import Container

agent = Container()

# 使用通义千问
response = agent.chat(prompt="你好", provider="qwen", type="medium")
# 或使用中文别名
response = agent.chat(prompt="你好", provider="通义", type="high")

# 使用智谱 GLM
response = agent.chat(prompt="你好", provider="zhipu", type="medium")
# 或使用别名
response = agent.chat(prompt="你好", provider="glm", type="high")

# 使用百川
response = agent.chat(prompt="你好", provider="baichuan", type="high")
```
