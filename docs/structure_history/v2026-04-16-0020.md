### 新增 Router 层 - 多 AI 提供商路由

基于 LiteLLM 实现统一的多提供商路由层，支持通过 `provider` 和 `type` 字段访问不同级别的模型。

### 4. 使用方式

```python
from container import Container

agent = Container()

# 方式 1: 使用 router.chat() 直接调用
response = agent.chat(
    prompt="你好",
    provider="anthropic",  # 可选，默认 anthropic
    type="medium",         # 可选，默认 medium (high/medium/low/auto)
)

# 方式 2: 智能路由
response = agent.chat(
    prompt="这是一个复杂问题...",
    type="auto",  # 根据输入长度自动选择层级
)
```
