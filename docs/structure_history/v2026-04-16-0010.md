## 变更说明

将日志级别配置移到 `src/config.py` 中统一管理。

### 修改的文件

#### `src/config.py`
- 删除 `debug: bool` 字段
- 新增 `log_level: str = "INFO"` 字段

#### `container.py`
- 导入 `src.config import config`
- 从 config 读取日志级别（可通过参数覆盖）

#### `main.py`
- 从 config 读取日志级别
- 更新提示信息

### 使用方式

修改 `src/config.py` 中的 `log_level`：

```python
class Config(BaseModel):
    # 日志级别 (ERROR, INFO, DEBUG)
    log_level: str = "INFO"  # 改为 "ERROR" 或 "DEBUG"
```

### 日志级别说明

| 级别 | 输出内容 |
|------|----------|
| ERROR | 只输出错误信息 |
| INFO | 输出方法调用名 |
| DEBUG | 输出参数、结果等详细信息 |
