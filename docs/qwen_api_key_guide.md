# Qwen API Key 获取指南

## 获取步骤

### 1. 访问阿里云百炼控制台

打开网址：https://bailian.console.aliyun.com/

### 2. 登录阿里云账号

- 如果没有账号，需要先注册阿里云账号
- 完成实名认证（需要中国大陆手机号）

### 3. 进入 API-KEY 管理

1. 在百炼控制台左侧菜单找到 **API-KEY 管理**
2. 或者在顶部导航栏点击你的账号 -> API-KEY 管理

### 4. 创建新的 API-KEY

1. 点击 **创建新的 API-KEY** 按钮
2. 输入 API-KEY 名称（如：AgentMemory）
3. 点击确认创建

### 5. 复制 API-KEY

- **重要**: API-KEY 只会显示一次，请立即复制保存
- 如果丢失，可以删除后重新创建

### 6. 配置到项目

1. 在项目根目录创建 `.env` 文件（复制 `.env.example`）
2. 填入你的 API-KEY：

```bash
# .env
QWEN_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxx
```

## 常见问题

### Q: API-KEY 格式是什么？
A: 通常以 `sk-` 开头，后跟一串字母数字

### Q: API-KEY 会过期吗？
A: 不会过期，除非手动删除或禁用

### Q: 如何限制消费额度？
A: 可在阿里云费用中心设置 **消费预警** 和 **额度限制**

### Q: 调用失败怎么办？
A: 检查以下几点：
1. API-KEY 是否正确复制（无多余空格）
2. 账户是否有足够余额
3. 是否完成实名认证
4. 查看错误信息，可能是请求格式问题
